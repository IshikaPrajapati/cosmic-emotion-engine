# Cosmic Emotion Engine

A hackathon project using **AWS Nova** and frontend JS to transform user emotions into cosmic visuals, sounds, and poetic reflections.

## Features
- Camera-based emotion detection or manual input
- Cosmic galaxy animation
- Emotion-based relaxation frequencies
- Poetic reflections generated by Amazon Nova

## Tech Stack
- Frontend: HTML, CSS, JS, face-api.js, Three.js (optional)
- Backend: Python Flask
- AI: Nova API for text generation
- Hosting: GitHub Pages / Localhost for frontend
